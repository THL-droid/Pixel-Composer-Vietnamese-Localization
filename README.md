# Pixel Composer v1.19 Việt hóa
Đây là kho lưu trữ chứa các tệp bản địa hóa Tiếng Việt dành cho Pixel Composer
Các tệp bao gồm là words.json và nodes.json (hiện tại là nodes.json vẫn chưa được dịch)
# Phiên bản tương thích
Phiên bản v1.19.0.9 hoặc cũ hơn

# Tệo bao gồm
words.json
nodes.json

# Cách cài đặt
Sao chép thư mục "vn" vào thư mục cài đặt Pixel Composer
Mở phần mềm → ở thanh trên cùng chọn Help > Open local directory → vào thư mục Locale rồi dán thư mục "vn" vào.
Sau đó khởi động lại phần mềm, mở File > Preferences > Interface > Interface Language, rồi chọn "vn".
Khởi động lại lần nữa để áp dụng bản Việt hóa.

# Gỡ cài đặt
Mở phần mềm → ở thanh trên cùng chọn Help > Open local directory → vào thư mục Locale rồi xóa thư mục "vn".
Sau đó khởi động lại phần mềm, mở File > Preferences > Interface > Interface Language, rồi chọn lại "en".
Khởi động lại lần nữa để quay về bản tiếng Anh.
